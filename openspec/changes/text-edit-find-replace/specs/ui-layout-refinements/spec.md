## MODIFIED Requirements

### Requirement: Top-Left UI Menu
The system SHALL display the UI control panel in the top-left corner of the screen. The panel SHALL include sections for text input, file upload, split methods, node limits, and text editing tools (Find/Replace).

#### Scenario: UI Layout with Editing Tools
- **WHEN** the application loads
- **THEN** the `#ui` menu contains labeled input fields for "Find" and "Replace", and a "Replace All" button.
