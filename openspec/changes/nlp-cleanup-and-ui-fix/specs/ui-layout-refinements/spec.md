## ADDED Requirements

### Requirement: Contained Find and Replace UI
The system SHALL ensure that the "Find and Replace" block in the control menu does not overflow its parent container. The layout must be responsive and stay within the menu bounds even when nested or when labels are long.

#### Scenario: Find and Replace within Menu
- **WHEN** the user opens the control menu
- **THEN** the "Find" and "Replace" input fields and their labels are fully visible and do not extend beyond the horizontal boundaries of the menu.
